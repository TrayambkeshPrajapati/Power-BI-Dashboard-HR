## 📌 Project Overview

The **HR Analytics Dashboard** is an interactive Power BI project designed to analyze employee workforce data and provide meaningful insights into employee performance, engagement, attrition, and compensation.

The dashboard transforms raw HR data into a clean, interactive reporting solution using **ETL processes, data modeling, and DAX calculations**.

The report includes both **Light Theme and Dark Theme dashboards** with a modern UI design and page navigation for better user experience.

---

# 🎯 Business Objectives

The dashboard helps HR teams and management answer important questions:

- How many employees are currently working?
- What is the employee attrition rate?
- Which departments have the highest workforce?
- How is employee performance distributed?
- What factors influence employee turnover?
- How does salary vary across departments?
- What is the employee engagement and satisfaction level?

# 🔄 Data Cleaning & ETL Process

The raw HR dataset was transformed using Power Query.

### Data Preparation Steps:

✅ Removed duplicate records  
✅ Handled missing values  
✅ Corrected inconsistent data types  
✅ Standardized date formats  
✅ Cleaned text fields  
✅ Created calculated columns  
✅ Prepared data model for reporting
